Cours : données structurées

Introduction : les données constituent la matière première de toute activité numérique. Différents types de données peuvent être crées et stockées sous divers formats. Elles sont le plus souvent structurées pour être facilement traitées. De nouveaux modes de stockage se développent depuis quelques années grâce à des avancées technologiques et numériques. I. Les types de données et leurs formats Une donnée est une valeur Les données sont une représentation d’une information au sein d’un système informatique. Elles peuvent être personnelles, permettant d’identifier une personne. Les données se déclinent en différents types de données ou format : image, texte, tableau, vidéo…. Le format de données correspond à la manière dont cette donnée est codée. Chaque format est identifié par une extension = .doc, .exe, .mp3, .jpeg, .pdf…correspondant aux logiciels qui les créent. Ces données si elles ne sont pas organisées sont qualifiées de données brutes. Mais pour les utiliser plus aisément, elles vont être structurées. Des données peuvent être crées automatiquement dès qu’elle s’enregistre : ce sont des métadonnées qui précisent des caractéristiques de la donnée (nom, lieu, heure, appareil utilisé, durée, taille, titre, accès….). Le premier support de données était la carte à trou puis la disquette qui a actuellement disparu mais dont le logo représente l'enregitrement dans toute interface numérique. Les données possèdent plusieurs supports de stockage : - internes : disque dur, carte SSD - externes : clé USB, disque dur externe - externes et distants : le Cloud Le disque dur dont la capacité de stockage était importante est supplanté depuis peu par les cloud, serveurs externes et distants dont les capacités sont plus importantes égales. II. Structuration de données et leurs traitements A. Données structurées et formats de fichiers structurés Les données sont structurées dans des tables ou tableaux ou collection. La première ligne précise les descripteurs, les lignes suivantes indiquent les données, séparées par des virgules (ou point virgule). Ils sont enregistrés dans des fichiers au format CSV (Common separated values = données séparées par des virgules) ou JASON.
